Planner Conditions Tested
ambient_prompt
Definition: The LLM receives both benign memory and poison memory. This is the unsafe baseline condition showing what happens when memory is passed directly into context.

capsule_filtered_prompt
Definition: The LLM receives only policy-authorized capsules after the capsule filter removes unauthorized memory. This is the defended condition.

jailbreak_style_prompt
Definition: The LLM receives benign memory plus a poisoned memory containing direct jailbreak-style planning instructions. This tests whether visible malicious memory can steer the LLM.
