Medium Live LLM Workflow-Corpus Case List
Generated from: data\workflow_corpus_splits\test.jsonl

Purpose
This file lists the 36 workflow-corpus cases used by the medium live LLM benchmark.
The benchmark runs each case under two planner conditions:
ambient_prompt and capsule_filtered_prompt.

Medium command
python run_llm_experiment.py --provider ollama --models llama3,mistral,phi3 --case-source workflow-corpus --workflow-corpus data\workflow_corpus_splits\test.jsonl --case-limit 36 --case-seed 2026 --repetitions 1 --output-csv results\medium_live_llm_planner_suite.csv --summary-csv results\medium_live_llm_planner_summary.csv --model-summary-csv results\medium_live_llm_planner_model_summary.csv

Scale
36 cases x 2 planner conditions x 3 models x 1 repetition = 216 live planner rows.
With --repetitions 3, this becomes 648 live planner rows.

Columns

attack_type: poisoning or benign workflow family.
poisoned: whether at least one event is marked as poisoned.
expected: safe expected recommendation.
bad: attacker target recommendation when poisoned.
